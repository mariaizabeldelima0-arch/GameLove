# 🏗️ Alan's Career Game - FG Empreendimentos

Um jogo interativo de plataforma 2D que celebra a jornada de carreira de Alan na área de segurança do trabalho.

## 🎮 Como Jogar

Acesse o jogo aqui: **[Alan's Career Game](https://mariaizabeldelima0-arch.github.io/GameLove/)**

Ou execute localmente:
```bash
python3 -m http.server 8000
# Abra http://localhost:8000 no navegador
```

## 📋 Fases de Carreira

### ✅ Fases Desbloqueadas

1. **🌱 Estagiário** (17/09/2025)
   - Dificuldade: Fácil
   - Desafios: Plataforma básica + 2 perguntas de segurança

2. **🧰 Auxiliar de Técnico de Segurança**
   - Dificuldade: Médio
   - Desafios: Plataforma com obstáculos + 2 quizzes + Nó 8 Simples

3. **👷 Assistente de Técnico de Segurança** (26/06/2026 - Promoção Recente!)
   - Dificuldade: Difícil
   - Desafios: Plataforma avançada + 3 quizzes + Nó 8 Duplo

### 🔒 Fase Bloqueada

4. **🏆 Técnico de Segurança**
   - Status: Bloqueada até conclusão do curso SENAI
   - Dificuldade: Expert
   - Desafios: Máximo nível de dificuldade

## 🎯 Objetivos de Cada Fase

Para completar uma fase, você precisa:

1. ⏃ **Navegar pela Plataforma** - Chegue ao final do nível evitando obstáculos
2. 🧠 **Responder Quiz de Segurança** - Acerte pelo menos 70% das questões
3. 🪢 **Desafio de Nós** (Fases 2-3) - Desenhe os nós corretamente

## 📚 Tópicos de Segurança Abordados

- **EPI** - Equipamento de Proteção Individual
- **NR-6** - Norma Regulamentadora de Equipamentos
- **CIPA** - Comissão Interna de Prevenção de Acidentes
- **PCMSO** - Programa de Controle Médico de Saúde Ocupacional
- **NR-35** - Proteção contra quedas em altura
- **ART** - Anotação de Responsabilidade Técnica
